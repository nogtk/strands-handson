# Amazon Bedrockで利用可能なモデルプロバイダー

Amazon Bedrockは、AWSが提供する完全マネージド型のサービスで、様々な基盤モデル（Foundation Models）を簡単に利用できるプラットフォームです。以下のモデルプロバイダーのモデルを利用できます。

## 主要なモデルプロバイダー

- **Amazon** - テキスト、画像、ドキュメント、ビデオ理解、画像・ビデオ生成、インタラクティブな音声、コード生成などに対応したTitanモデルファミリーを提供
- **Anthropic** - 複雑な推論、コード生成、指示への対応に優れたClaudeモデル
- **DeepSeek** - 複雑な問題を段階的に解決する高度な推論モデル
- **Mistral AI** - エージェント推論やマルチモーダルタスクに特化した専門家モデル
- **Meta** - 高度な画像・言語理解を提供するLlama 2などのモデル
- **OpenAI** - コーディング、科学的分析、数学的推論に優れたオープンウェイトモデル

## その他の利用可能なプロバイダー

- **AI21 Labs** - Jurassicモデルを提供
- **Cohere** - CommandとEmbedモデルを提供
- **Stability AI** - Stable Diffusionなどの画像生成モデルを提供
- **Luma AI** - ビジュアル生成モデルを提供
- **Qwen** - テキスト生成モデルを提供
- **TwelveLabs** - ビデオ理解モデルを提供
- **Writer** - テキスト生成モデルを提供

## Amazon Bedrock Marketplace

Amazon Bedrockでは、「Amazon Bedrock Marketplace」を通じて100以上の基盤モデルにアクセスすることができます。このマーケットプレイスでは、以下のようなモデルプロバイダーからモデルを利用できます：

- **IBM** - Graniteモデルなどを提供
- **Nvidia** - 様々なモデルを提供
- **Upstages** - 特化型モデルを提供
- **Evolutionary Scale** - 特化型モデルを提供
- **Arcee AI** - 特化型モデルを提供
- **Widn.AI** - 特化型モデルを提供
- **Zendesk** - 翻訳ユースケース向けの特化型モデルを提供

## 新たに追加されたモデル

最近、Amazon Bedrockには18の新しいモデルが追加されました。特に注目すべきは以下のMistral AIのモデルです：

1. **Mistral Large 3** - 長文脈理解、マルチモーダル、指示信頼性に最適化
2. **Ministral 3 3B** - エッジ最適化、単一GPU対応の言語・ビジョン機能を提供
3. **Ministral 3 8B** - 単一GPU対応で高パフォーマンスと最小フットプリントを提供
4. **Ministral 3 14B** - 単一GPU対応で最先端のテキストとビジョンのパフォーマンスを提供

## モデルの利用方法

Amazon Bedrockのモデルは以下の方法で利用できます：

1. 推論の実行 - プロンプトを送信してレスポンスを生成
2. モデル評価 - 出力を比較し、ユースケースに最適なモデルを決定
3. ナレッジベース構築 - 埋め込みモデルを使用してクエリに対するレスポンスを生成
4. エージェント作成 - モデルを使用してプロンプトに対する推論を実行
5. モデルカスタマイズ - トレーニングおよび検証データを提供してモデルパラメータを調整
6. プロビジョンドスループットの購入 - スループットを向上

## まとめ

Amazon Bedrockは、幅広いモデルプロバイダーからの基盤モデルを単一のAPIから簡単にアクセスできるようにすることで、ユーザーが様々なユースケースに対応するAIアプリケーションを構築する柔軟性を提供します。
